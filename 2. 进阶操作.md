# 进阶操作

## 回滚功能

按两下Esc或输入/rewind进入回滚界面，选择回滚的聊天和代码

但是这种方法只能用于回滚CC自己编写的代码，如果是CC执行的一些终端命令，如安装了一些包，那是回滚不了的。

最好的办法是用git：

首先和cc说：`帮我下载git，并和我的github账号绑定`

按照cc的指引一步步做下去即可

然后和cc说：`帮我把XXX这个文件夹提交到远程仓库中去`

这样，就可以在更新代码之后，可以在cmd中将代码更新到github的仓库中: `cd d:/Code/ClaudeCodeLearning && git push`

或者通过聊天的方式和cc说：`把本地代码推送到 GitHub`

如果遇到需要回滚的情况，直接和cc说：`帮我回滚到Github存档的版本`

当代码有了新版本，在不想影响原版本的基础上建立新的版本，可以通过创建新的分支（branch）实现：`将本地代码以一个新的分支的形式推送到Github`
