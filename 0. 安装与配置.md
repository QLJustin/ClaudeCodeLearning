# 安装与配置

## 1. 安装Vscode

略

## 2. 安装Claude Code

在Vscode中安装Claude Code插件

取官网上将Claude Code安装在系统上

## 3. 下载CC switch

CC switch 可以帮助我们轻松地管理各种模型

## 4. 配置模型

进入网页版的deepseek的api配置界面，建立api并复制到cc switch中。

配置模型：

- 主模型：deepseek-v4-flash
- Haiku：deepseek-v4-flash
- Sonnet：deepseek-v4-flash[1m]
- Opus：deepseek-v4-pro[1m]

解释：主模型是默认模型，Haiku是性能最弱的模型，Sonnet是均衡模型，Opus是性能最强也最贵的模型。
